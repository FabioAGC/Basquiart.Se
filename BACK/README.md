### Rodando o projeto ###
certifique-se que o docker esta instalado e rodando, ele sera necessario para baixar o postgress

rode o installDependencies.bat e espere, isso so sera necessario rodar uma vez ao baixar o repositorio

apos isso rode RunServer.bat para rodar o servidor, ele sera servido em 127.0.0.1:8000
